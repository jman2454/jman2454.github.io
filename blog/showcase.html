    • Divs:

    For this kind of div, do a div with the class "Main".
    For the kind found on the 404 page, do a div with the class "big".
    For a blog div, do a div with the class, "blog" and then where the post goes do another div with the class "blogcontainer"

    Navbar:

    1. Make a div with class 'navbar'
    2. Make an li with class 'navbar' in the div
    3. Make an unordered list
    4. Make multiple li's and the links with class 'nav' (for the home page, you can do class 'header'

    That's Basically it!

    I'll add more cool features soon, but enjoy what is up so far!

    Alerts:

    So far there is only 1 type of alert. It is on the homepage.
    To make one of these, make a div with the class 'alert alert-success'. Then, make a strong tag < strong > (remove spaces). Type what you want in the strong tag for the bold part. Then close the strong tag and type the rest of the alert message.

    Buttons:

    There are 4 colors of buttons, and 3 sizes. To make a button, first either make an 'a' tag or 'button' tag with class btn. If you want the button to be green, do 'btn btn-green' as the class. The other colors are orange and red. For a large button, add 'lg' to the class. for medium, change nothing, and for small, add 's' to the class. Here is an example small orange button.
